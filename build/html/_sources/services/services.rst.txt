ID Card Replacement
====================

Before requesting for ID card replacement, please give yourself time to ensure that you really need ID card replacement. 
ID replacement section allows student to request a new ID card and view the status of the lodged request. Certain request will require one to have attach evidence, example a police loss report, and each request must have a request reason. 

.. image:: ../images/idreplacement.png
